# React Admin Panel

## Setup

1. Navigate to the `admin-panel` directory:
```bash
cd admin-panel
```
2. Install dependencies:
```bash
npm install
```
3. Update `src/firebaseConfig.js` with your Firebase project config.
4. Start locally:
```bash
npm start
```
5. Build for production & deploy:
```bash
npm run build
firebase deploy --only hosting
```
